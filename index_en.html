M3U8Player is a Chrome extension that plays .m3u8 HTTP Live Streaming (HLS) content. It uses HTML5 and Media Source Extensions to decode and play videos. MPEG-2 transport streams and AAC/MP3 audio are converted into ISO BMFF (MP4) segments. Web Worker support improves performance with asynchronous decoding.

What is an M3U8 file?

An M3U8 file is a UTF-8 encoded audio playlist file. It is a plain text file that helps media players locate audio or video files.

For example, an M3U8 file might point to an online radio stream or a video playlist on your local device.

You can open the file and instantly start streaming the content it refers to. If you enjoy the same media regularly, you can create an M3U8 playlist for quick access.

M3U8 files support absolute/relative paths or URLs and may contain comments describing the playlist.

Although originally intended for audio, M3U8 is widely used today for video streaming. Many players support the M3U8 format.
